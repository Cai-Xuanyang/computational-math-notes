# Computational Mathematics Notes

This repository contains professional notes on mathematical writing, featuring LaTeX typesetting with theorem environments, colored proof boxes, and proper mathematical formatting.

## Contents

- Mathematical writing principles and techniques
- Analysis of proof styles and common errors
- Professional typesetting with tcolorbox and amsmath packages
- Automated build and spell checking workflows

## Building Locally

# Build PDF
latexmk -pdf -interaction=nonstopmode main.tex

# Clean temporary files
latexmk -c
